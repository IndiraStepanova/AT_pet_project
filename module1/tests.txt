Пользовательский сценарий 1.1. Регистрация нового пользователя
Тестовый сценарий 1.1.1. Регистрация нового пользователя
Предусловия: открыта страница регистрации/авторизации http://selenium1py.pythonanywhere.com/ru/accounts/login/
1. В блоке "Зарегистрироваться" ввод в поле "Адрес электронной почты" адреса электронной почты
2. Ввод и подтверждение пароля удовлетворяющего требованиям
3. Нажать кнопку "Зарегистрироваться"

Ожидаемый результат:
1. Поля ввода активны
2. Регистрация произведена
3. На указанный адрес электронной почты пришла ссылка для подтверждения регистрации

Пользовательский сценарий 2. Авторизация ранее зарегистрованного пользователя
Тестовый сценарий 1.2.1. Авторизация ранее зарегистрованного пользователя
Предусловия: открыта страница регистрации/авторизации http://selenium1py.pythonanywhere.com/ru/accounts/login/
1. В блоке "Войти" указать указанный в тестовом сценарии 1.1.1. адрес электронной почты
2. Указать пароль указанный в тестовом сценарии 1.1.1.
3. Нажать "Войти"

Ожидаемый результат:
1. Поля ввода активны
2. Авторизация пользователя произведена, пользователь успешно вошел на сайт под своей учетной записью

Тестовый сценарий 2.1.1.Выбор товаров в корзину
Предусловия: открыта главная страница http://selenium1py.pythonanywhere.com/ru/catalogue/

Пользовательский сценарий 2.2.Оформление покупки
Тестовый сценарий 2.2.1. Оформление покупки
Предусловия: открыта главная страница http://selenium1py.pythonanywhere.com/ru/catalogue/
1. Отобрать несколько товаров в корзину
2. Пройти в корзину
3. Нажать "Перейти к оформлению" и заполнить форму для оформления доставки покупки
4. Нажать "Продолжить"
Ожидаемый результат:
1. Кнопка "Добавить в корзину" активна
2. В корзине товары и их количество соответствуют выбору пользователя
3. Кнопка "Перейти к оформлению" активна, после нажатия на нее происходит переход на страницу для оформления покупки
4. Все поля на странице оформления покупки активны, поля обязательные к заполнению отмечены звездочкой.
5. После заполнения всех полей и нажатия кнопки "Продолжить" происходит переход на страницу оплаты.
